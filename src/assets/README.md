# 静态资源目录

此目录用于存放应用的静态资源文件。

## 目录结构

```
assets/
├── images/          # 图片资源
├── icons/           # 图标文件
├── scenes/          # 场景背景图
├── avatars/         # 虚拟形象资源
└── fonts/           # 自定义字体
```

## 使用说明

### 图片资源
- 支持格式：PNG, JPG, SVG, WebP
- 建议尺寸：根据使用场景优化
- 命名规范：使用小写字母和连字符，如 `scene-cafe.jpg`

### 图标
- 推荐使用 SVG 格式
- 保持简洁清晰
- 支持多种尺寸

### 场景背景
- 建议尺寸：1920x1080
- 格式：JPG 或 WebP（压缩优化）
- 每个场景至少准备一张主图

### 虚拟形象
- 支持 2D/3D 模型资源
- 包含表情、动作等动画资源
- 格式根据引擎要求

## 注意事项

1. 所有资源文件应进行压缩优化
2. 大图使用懒加载
3. 考虑移动端网络环境
4. 遵循版权规定
