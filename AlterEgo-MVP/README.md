# Alter Ego - AI 多元场景口语实战 MVP

> 重塑你的英语第二自我 —— 在安全的虚拟世界中，自信地开口说英语

## 产品概述

Alter Ego 是一款结合心理学与 AI 技术的游戏化英语口语训练应用。通过"理想自我投射 + 场景化沉浸 + 思维矫正机制"三大核心创新，帮助用户突破"哑巴英语"与"中译英思维"双重障碍。

### 核心功能

- **理想自我投射** - 创建你的虚拟形象，在安全的环境中练习
- **多元场景实战** - 咖啡厅、机场、会议室...真实场景模拟
- **思维矫正** - 切断翻译思维，建立英语直觉反应
- **三级递进提示** - 语音提示 → 关键词提示 → 中文翻译
- **重述式隐性纠错** - AI 用正确表达自然重述，不直接纠正错误

## 快速开始

### 运行方式

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 开始体验！

```bash
# 使用 Python 启动本地服务器（可选）
cd AlterEgo-MVP
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问 `http://localhost:8000`

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **头像服务**: DiceBear API
- **数据存储**: localStorage

## 功能说明

### 1. 形象创建

- 支持 2D 冒险风和 3D 卡通风两种风格
- 6 种主题颜色可选
- 16 个预设形象快速选择
- 形象数据本地持久化存储

### 2. 场景练习

- **咖啡厅** - 点单、闲聊的日常场景（2 星难度）
- **机场** - 值机、过海关的旅行场景（3 星难度）
- **会议室** - 商务会议、工作汇报（4 星难度）

### 3. 对话系统

- 按住说话，松开发送
- 实时录音状态显示
- AI 自动响应（模拟重述法）
- 对话历史滚动显示

### 4. 提示系统

| 等级 | 类型 | 说明 |
|------|------|------|
| 1 | 语音提示 | 慢速重复关键词 |
| 2 | 关键词提示 | 首字母/高亮显示 |
| 3 | 中文翻译 | 完整翻译参考 |

### 5. 学习统计

- 总练习时长
- 练习次数
- 连续打卡天数
- 学习进度可视化
